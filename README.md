# AI-Powered Resume Screener

An AI-powered resume screening system built using Python, LangChain, and FAISS to enable semantic candidate matching beyond traditional keyword-based filtering.

## Features
- Semantic resume matching
- Multi-format parsing (PDF, DOCX, JSON)
- Vector-based candidate retrieval using FAISS
- Automated ranking of candidate profiles

## Tech Stack
- Python
- LangChain
- FAISS
- NLP
- JSON

## Project Overview
The system processes resumes from multiple file formats, converts candidate information into vector embeddings, and retrieves the most relevant profiles based on semantic similarity to job requirements.
