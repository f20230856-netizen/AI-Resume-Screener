# AI-Powered Resume Screener

An AI-driven resume screening system built using Python, LangChain, and FAISS for semantic candidate matching.

## Features

- Semantic resume matching
- Multi-format resume parsing
- FAISS vector search
- Candidate ranking
- JSON-based output generation

## Tech Stack

- Python
- LangChain
- FAISS
- NLP
- JSON

## Project Structure

resume_screener/
│
├── sample_resumes/
├── document_loader.py
├── embed_and_query.py
├── main.py
└── README.md

## Usage

1. Add resumes to sample_resumes/
2. Run main.py
3. Enter job description
4. View ranked candidates
